# 爱运动官网
isport

#### 介绍
爱运动官网，包括Logo、标题、描述、下载地址、宣传图、关于我们、广告合作、备案等内容展示。
支持web和h5页面。

#### 软件架构
jquery
插件包括：jquery.fullPage.min.js、jquery.bxslider.js

#### 安装教程
1. cd workspace
2. git clone https://gitee.com/chong2230/isport.git

#### 使用说明
下载项目后，点击index.html即可运行。
